Instalando eLua na família de CPUs STR7 da ST

STR7 é uma família de processadores baseados no ATM7TDMI da ST. Eles são pequenos, MCUs de baixa potência, com um conjunto equilibrado de periféricos embutidos no chip. Estou usando a placa MOD711 da ScTec. A placa está baseada na variante STR711FR2 da família STR7. Uma vez que esta placa ainda está em desenvolvimento, fui obrigado a adicionar algumas coisas: um conversor MAX3232 RS232 para TTL para a interface serial, um par de LEDs e um botão de reset. Depois disso, a placa estava pronta para eLua:)

Prá-requisitos

Antes de começar a usar eLua na CPU STR711FR2, certifique-se de que:

  • você está usando Linux, Windows, ou qualquer outro sistema operacional que tenha suporte para OpenOCD. Você pode dar uma olhada no meu tutorial do OpenOCD antes de continuar.
  • você já tem o seu arquivo com a imagem de eLua para a CPU STR711FR2 (gerado ou baixado).

Gravando eLua na placa MOD711

Você precisa do OpenOCD para fazer isso. Basta seguir as instruções do meu tutorial OpenOCD. Na página do tutorial, você também encontrará links para os arquivos de configuração do OpenOCD que estou usando para gravar eLua na placa MOD711. E é isso aí! eLua está gravado na CPU, agora é só iniciar o emulador de terminal e se divertir, conforme descrito em using eLua.

NOTA IMPORTANTE: para esta placa, você precisa configurar a velocidade de sua porta serial para 38400 (ao contrário de 115.200 baud para as outras placas). Todos os outros parâmetros continuam os mesmos (8 bits de dados, sem paridade, um stop bit).
